# CrossZero

## Project Description

CrossZero is a graphical implementation of the classic Tic-Tac-Toe game written in Python using the Turtle graphics library. The project demonstrates event-driven programming, graphical rendering, and game logic in a simple desktop application.

## Installation and Running Instructions

1. Clone the repository

   ```bash
   git clone https://github.com/rostikprogrammer228/CrossZero.git
   ```

2. Navigate to the project folder

   ```bash
   cd CrossZero
   ```

3. (Optional) Create a virtual environment

   **Windows**

   ```bash
   python -m venv venv
   ```

   **macOS/Linux**

   ```bash
   python3 -m venv venv
   ```

4. Activate the virtual environment

   **Windows**

   ```bash
   venv\Scripts\activate
   ```

   **macOS/Linux**

   ```bash
   source venv/bin/activate
   ```

5. No additional dependencies are required.

6. Run the project

   ```bash
   python CrossZero.py
   ```

## Project Summary

CrossZero is a Python implementation of Tic-Tac-Toe that demonstrates event handling, graphical programming with Turtle, coordinate-based game logic, and victory detection.
